# House Price Prediction  
This project predicts house prices using machine learning.  
## Tools Used:
 - Python
 - Pandas, NumPy
 - Scikit-learn
 - Matplotlib
## Model: 
 - Linear Regression
## Features: 
- id
- Date
- number of bedrooms
- number of bathrooms
- living area
- lot area
- number of floors
- waterfront present
- number of views
- condition of the house
- grade of the house
- Area of the house(excluding basement)
- Area of the basement
- Built Year
- Renovation Year
- Postal Code
- Lattitude
- Longitude
- living_area_renov
- lot_area_renov
- Number of schools nearby
- Distance from the airport
## Result:
 Achieved R2 Score of ~0.7678739802736996
